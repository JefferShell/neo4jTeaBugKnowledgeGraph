## 项目名称：知识图谱问答系统

### 项目简介：

这个项目是一个基于知识图谱的问答系统，可以从电影相关的知识图谱中回答用户的问题。系统主要功能包括：

- 基于图谱的知识推理，回答与电影相关的各种问题。
- 图谱展示，可可视化电影之间的联系。
- 基于模板匹配的自动回答，并根据知识图谱内容进行深度推理。


### 主要功能：

- 图谱问答
- 图谱展示
- 自动回答


### 技术架构：

- 前端：HTML、CSS、JS
- 后端：Django
- 数据库：Neo4j、SQLite
- 图谱工具：eCharts


### 使用说明：

1. 下载项目并解压缩。
2. 进入到根目录下的 xy_neo4j目录，并执行 `python build.py` 命令。
3. 等待数据初始化完成后，可以使用系统。
